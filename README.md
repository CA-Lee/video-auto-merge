# video-auto-merge
```
./合片自動機.sh list.txt
```

list.txt 格式長這樣:

```
"輸出檔名1.mp4"
"要合併的影片1.MTS"
"要合併的影片2.MTS"
---
"輸出檔名2.mp4"
"要合併的影片3.MTS"
"要合併的影片4.MTS"
---
```

必需以```---```結尾
